7. 问题
    下面 “a” 打印的值是多少？
    ```
    <?php
    $a = 1;
    function Test()
    {
        echo "a = $a";
    }
    Test();
    ?> 
    ```   
    函数体内引用全局变量，必须在函数内部使用“global”关键字(没有值)
8. 问题
    下面代码执行结果是？
    ```
    <?php
    FUNCTION TEST()
    {
        ECHO "HELLO WORLD!\n";
    }
    test();
    ?> 
    ```
    用户定义的函数和语言关键字对大小写不敏感。
9. 问题
    使用 PHP 创建注释的方法下面哪一种正确？
    ```
    // 注释写在这里
    /* 注释写在这里 */
    # 注释写在这里
    以上全部正确
    ``` 
    这三个都是有效的，它们是从 C、C++ 和 shell 代码中借用的。
